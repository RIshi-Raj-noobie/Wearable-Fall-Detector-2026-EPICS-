# 🧠 Wearable Fall Detection System (IoT + AIML)

## 📌 Overview
This project implements a smart wearable fall detection system designed for elderly safety using IoT and signal processing techniques.

## 🚀 Features
- Real-time fall detection using accelerometer & gyroscope
- Signal Vector Magnitude (SVM) based detection
- Post-fall immobility verification
- IoT-based cloud alert system
- Modular and scalable architecture

## 🏗️ Architecture
1. Sensor Layer (MPU6050)
2. Processing Layer (Feature Extraction)
3. Detection Layer (Hybrid Algorithm)
4. Communication Layer (WiFi)
5. Cloud Layer (API + Database)
6. Alert System (Buzzer + Mobile Notification)

## 🧪 Technologies Used
- Python
- NumPy
- IoT Concepts
- Signal Processing

## 📊 Future Improvements
- Machine Learning (SVM / LSTM)

- Real Hardware Deployment (ESP32)
